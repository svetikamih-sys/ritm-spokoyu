# Метаболічний Ритм — Pomodoro для зниження кортизолу

Цей інструмент створений не для "успішного успіху", а для жінок 45+, які хочуть домовитися зі своїм тілом. Це MVP таймера, який допомагає знизити рівень стресового гормону кортизолу через правильний ритм роботи та відпочинку.

## Чому це працює?
Для жінки 45+ хронічний стрес — це головний ворог стрункості. Коли ми працюємо "на виснаження", організм отримує сигнал запасати воду та жир. Цей таймер змінює правила гри:

* **Цикли по 25 хвилин:** Оптимальний час для фокусу без перевантаження нервової системи.
* **Антистрес-підказки:** Кожна перерва супроводжується порадою щодо розслаблення лімфи, дихання або зниження напруги [cite: 2026-01-25].
* **Трекер зволоження:** Нагадування про пиття води для боротьби з набряками [cite: 2026-01-10].

## Технології
* Чистий **HTML5 / CSS3 / JavaScript**.
* Адаптивний дизайн (Mobile First).
* LocalSearch для збереження статистики сесій та випитої води.

## Як запустити
1. Скачай файл `index.html`.
2. Відкрий його у будь-якому браузері.
3. Додай у закладки на телефоні або ноутбуці.

## Про автора
Консультант із харчування, незалежний партнер Herbalife. Допомагаю жінкам 45+ зрозуміти причини зупинки ваги та повернути ресурсний стан без дієт і тиску.
